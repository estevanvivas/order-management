\# Order Management System



\*\*Ingeniería de Software II\*\*



Este repositorio contiene dos versiones de un sistema de gestión de usuarios, productos, pedidos y pagos, desarrolladas como parte de un taller sobre arquitectura monolítica. El objetivo es ilustrar la diferencia entre un monolito acoplado y una arquitectura modular, analizando el impacto de las decisiones arquitectónicas sobre el acoplamiento y la cohesión del sistema.
